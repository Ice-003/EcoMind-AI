# 🧠 EcoMind AI - Sustainable Intelligence Starts Here

**EcoMind AI** is an AI-powered Sustainability Intelligence Platform that helps users optimize prompts for Large Language Models (LLMs). The platform analyzes prompt efficiency, estimates token waste, predicts cost impact, measures environmental footprint, and generates optimized prompts that achieve the same objective with fewer tokens, lower costs, and reduced carbon emissions.

Built as a full-stack AI application, EcoMind AI combines intelligent prompt engineering, sustainability analytics, and a modern user experience powered by Streamlit and Groq-hosted open-source LLMs.

---

## 🌟 Features

### 🚀 Prompt Intelligence Engine

* Analyze prompts across multiple optimization modes:

  * Concise
  * Balanced
  * Maximum Efficiency
* Preserve intent while reducing unnecessary tokens.
* Generate high-performance optimized prompts.

### 📊 Token Intelligence

* Compare original vs. optimized token usage.
* Calculate token savings.
* Visualize efficiency improvements.

### 💰 Cost Intelligence

* Estimate AI inference costs.
* Display costs in USD and INR.
* Track cumulative cost savings.

### 🌱 Carbon Intelligence

* Estimate energy consumption.
* Calculate carbon emissions generated by AI usage.
* Measure environmental savings achieved through optimization.

### 📈 Smart Analytics Dashboard

* Interactive visualizations using Plotly.
* Monitor:

  * Total prompts optimized
  * Tokens saved
  * Cost savings
  * Carbon savings

### 🗄️ Prompt Memory

* SQLite-powered storage.
* Search and review optimization history.
* Reuse previous optimized prompts.

### 📄 PDF Report Generator

* Export optimization results.
* Generate professional sustainability reports.
* Download reports for presentations and documentation.

### ⚙️ Custom Settings

* Configure API keys.
* Adjust cost estimation parameters.
* Modify carbon calculation factors.

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Ice-003/EcoMind-AI.git
cd EcoMind-AI
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
```

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Create Environment File

Create a `.env` file in the project root:

```env
GROQ_API_KEY=your_groq_api_key_here
```

---

## 💻 Running the Application

Start the Streamlit application:

```bash
streamlit run app.py
```

Open:

```text
http://localhost:8501
```

### Usage

1. Open the Settings page.
2. Configure your Groq API key.
3. Navigate to the Optimizer tab.
4. Enter an inefficient prompt.
5. Select an optimization mode.
6. Click **Optimize Prompt**.
7. Review token, cost, and carbon savings.
8. Export results as a PDF report.

## 🎯 Problem Statement

Large Language Models often process inefficient prompts containing redundant instructions and unnecessary tokens. This increases:

* API costs
* Processing time
* Computational resources
* Energy consumption
* Carbon emissions

EcoMind AI addresses this challenge by intelligently optimizing prompts while preserving their original intent and quality.
